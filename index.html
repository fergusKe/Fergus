京鋐科技 - 前端工程師

從資策會畢業後的第一份工作，公司主要的業務是提供 POS 系統給廠商使用。由於客戶主要都是線下的服務，因此希望提供線上服務給既有的客戶。開發的網站大多是購物網站，在這段期間累積了很多切版經驗。

喜歡學習新知識，有空會參加讀書會。有參加過 HPX 的讀書會，前一陣子也有自己發起了線上讀書會，與其他人在線上一起學習 Udemy 上的 『Modern React with Redux』。React 課程結束後，又參與了線上深度＆機器學習讀書會，學習教材是『An Introduction to Statistical Learning』，而我負責導讀 Tree-Based Methods 的章節。

有參加過接棒啟蒙舉辦的 『職業獵人-敏捷式網站遊戲開發實戰營』 。職業獵人專案是希望能幫助學生尋找自己天賦，並且能夠幫助學生學習的線上答題遊戲。玩家可以選擇自己感興趣的職業角色，透過答題打怪的方式取得經驗值或寶箱，最後可以得到挑戰大魔王的機會。

大數據與數據分析的技術興起，開始有越來越多的應用出現，我覺得這應該會是個趨勢，所以平時也有在注意相關資訊。在2016年正好有機會參與智庫驅動舉辦的『D4SG』計畫，幫助非營利或政府單位將累積的數據發揮最大的價值。

我2016年參與的專案是與台北市家防中心合作的 『家庭暴力案件預警與風險管理』，希望達到「降低被害人再受暴風險」及「提高社工服務效能」兩個主要目的。正好利用自己網頁開發的專長，使用視覺化的方式製作了風險地圖(案件數越多的區域，地圖顏色就會越深)。因為這是個學習以及協助社會的好機會，所以2017年又參加了一次，這次合作的單位是『均一教育平台』，他們是線上教育平台，目標是解決台灣教育資源不平等的問題，找出哪些因素會影響學生的學習成效。

2019年5月的時候，受到國立臺北護理健康大學資管系祝國忠老師的邀請，有機會到他們系上做一整天六小時的 Python Flask 網頁開發教學 (1) (2) 。
